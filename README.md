# Green Expense – Personal Finance Management Platform

## Overview

Green Expense is a full-stack financial management platform that helps users track their expenses, savings, debts, and upcoming bills in one place.
The application provides a clean dashboard for managing personal finances and setting financial goals.

The system is designed to simplify daily financial tracking and improve budgeting habits through organized expense records and bill reminders.

---

## Features

* Expense tracking and categorization
* Savings management
* Debt tracking system
* Monthly bill reminder calendar
* Clean financial dashboard
* Secure backend with environment-based configuration
* Global state management using Redux Toolkit

## Installation

### Clone the repository

```
git clone https://github.com/kaoushikkalyaannammi-7/Green-Expense.git
```

### Navigate to the project

```
cd Green-Expense
```

---

### Install backend dependencies

```
cd backend
npm install
```

---

### Install frontend dependencies

```
cd ../frontend/react
npm install
```

---

## Running the Application

### Start backend

```
cd backend
npm start
```

### Start frontend

```
cd frontend/react
npm start
```

---

## Environment Variables

Create a `.env` file inside the backend directory:

```
PORT=
DATABASE_URL=
JWT_SECRET=
```
